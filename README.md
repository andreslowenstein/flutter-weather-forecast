# WeatherForecast

WeatherForecast it's an weather app developed by Andrés Lowenstein.

## Usage:

### Install:
```sh
flutter pub get
```
### Run:
```sh
flutter build apk
```
Or you can run the app in your Android/IOS emulator directly.
